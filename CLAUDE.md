# CLAUDE.md

This project is a trading bot built in Python, used as a learning environment for Claude Code.

## Project

A Python-based trading bot. The goal is to learn Claude Code while building something functional.

## User Context

- The user is **new to programming** — always explain reasoning, not just actions
- Before writing or changing code, explain *why* this approach is being taken
- Define technical terms when they are introduced
- Walk through logic step by step so it is easy to follow and learn from

## Language & Style

- **Python** only
- Add **docstrings** to all functions and classes explaining what they do, their parameters, and return values
- Follow PEP 8 conventions
- Use type hints

## Code Guidelines

- Every function must have a docstring
- Keep functions small and single-purpose
- Prefer explicit over implicit
